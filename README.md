# 🌍 Air Quality Monitoring App

## Overview
The Air Quality Monitoring App is designed to monitor and display air quality information using pollutant data such as AQI, CO₂, PM2.5, and PM10 levels.

The application helps users understand environmental conditions and promotes awareness about air pollution.

---

## Features
- Real-time Air Quality Monitoring
- AQI Display
- Pollutant Analysis
- Interactive Dashboard
- Data Visualization
- Responsive Interface
- Environmental Alerts

---

## Technologies Used
- Python
- Streamlit
- APIs / Sensors
- Chart.js
